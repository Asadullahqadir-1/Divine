# Divine Platform Workspace

Monorepo structure:

- `studio/` - Sanity Studio v3 CMS
- `web/` - Next.js frontend integration layer (query/client stubs included)

## Quick Start

### Sanity Studio

```bash
cd studio
npm install
npm run dev
```

### Frontend (when ready)

The `web` folder currently contains Sanity integration scaffolding files so your Next.js app can plug in quickly.
